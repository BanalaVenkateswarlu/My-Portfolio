# 🛠️ Dual-Discipline Engineering Portfolio

Welcome to the repository for my professional portfolio website. This site is built to highlight the convergence of **Mechanical Engineering** principles and **Modern Software Engineering** implementations.

👉 **Live Demo:** [banala-venkateswarlu.netlify.app](https://banala-venkateswarlu.netlify.app/)

---

## 🚀 Key Features

*   **Adaptive Systems Design:** Responsive layouts driven by CSS Grid and Flexbox with zero layout shifts (CLS).
*   **Performance First:** Built purely with semantic HTML5, modular CSS3 custom variables, and clean vanilla ES6+ JavaScript. No bulky frameworks.
*   **Adaptive Themes:** Synchronizes automatically with system settings (dark/light mode) and retains manual overrides using `localStorage`.
*   **Accessibility (a11y):** Supports keyboard-only navigation via `:focus-visible` styling, structured semantic hierarchies, visual focus targets, and skip links.
*   **Dynamic Scroll Progress & Observers:** Seamless scroll-tracking using `IntersectionObserver` to highlight navigation links passively.
